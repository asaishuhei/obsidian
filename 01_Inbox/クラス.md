#program 

[オブジェクト指向](オブジェクト指向.md)で出てくる概念の一つ。

設計図に相当するもので、[プロパティ](プロパティ.md)と[メソッド](メソッド.md)を定義したもの。

クラスを設計して
```php
class Person {
  private $name;//プロパティ

  public function setName($name) {//メソッド
    $this->name = $name;
  }

  public function getName() {//メソッド
    return $this->name;
  }
}
```

クラスを呼び出して`$person`オブジェクトを作成
```php
$person = new Person();
$person->setName("John");
echo $person->getName(); // "John"
```
